## 5. Работа в Jupyter Notebook

### 5.1 Что такое ноутбуки и зачем они нужны
Jupyter Notebook — это интерактивная среда, где код, текст, графики и результаты выполнения живут в одном документе.  
Это делает ноутбуки удобными для:

- экспериментов и быстрых прототипов;
- анализа данных;
- обучающих материалов;
- визуализации результатов.

Ноутбук — это по сути веб-интерфейс к интерпретатору Python.

### 5.2 Где и как запускать ноутбуки
Запуск локально:
```bash
pip install jupyter
jupyter notebook
````

Современный вариант:

```bash
pip install jupyterlab
jupyter lab
```

Работа через IDE:

* VS Code умеет открывать `.ipynb` без дополнительной магии.
* Можно подключаться к удалённым серверам и окружениям.

Важный момент: ноутбук работает внутри **конкретного виртуального окружения**, поэтому выбранный интерпретатор определяет, какие библиотеки доступны.

### 5.3 Лучшие практики

* Делить документ на логические блоки.
* Фиксировать зависимости, чтобы воспроизводимость не страдала.
* Не превращать ноутбук в «магический дневник из 200 ячеек», который невозможно прочитать.
* Для проектов код лучше выносить в модули, а ноутбук использовать для экспериментов.
